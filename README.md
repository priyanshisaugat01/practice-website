# 🌐 Practice Website

This is a simple **static website project** built using HTML and CSS, and deployed using **AWS S3 static website hosting**.

👉 https://priyanshisaugat01.github.io/practice-website/
---

## 🛠️ Tech Stack

* HTML
* CSS
* AWS S3 (Static Website Hosting)
* GitHub

---

## 📦 Features

* Multi-page website
* Clean and simple UI
* Static deployment on cloud

---

## 📁 Project Structure

```
practice-website/
│── index.html
│── contact.html
│── menu.html
│── css/
```

---

## ☁️ Deployment (AWS S3)

Steps followed:

1. Created an S3 bucket
2. Uploaded website files
3. Enabled static website hosting
4. Configured `index.html`
5. Made files publicly accessible
6. Accessed via S3 endpoint
